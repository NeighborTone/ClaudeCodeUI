# /dev - Full Development Cycle

Run the complete development cycle: build, test, verify.

## Usage

```
/dev
```

## Workflow

1. **Build**: Run the project build command
2. **Test**: Execute automated tests (if available)
3. **Verify**: Check for errors and warnings

## Steps

1. Execute build command
2. Check build output for errors
3. Run test suite (if configured)
4. Report results

## Output Format

```
DEV CYCLE RESULTS:
- Build: [SUCCESS/FAILED]
- Tests: [PASS/FAIL/SKIPPED] (X/Y passed)
- Warnings: [count]
- Status: [READY/BLOCKED]
```
